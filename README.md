I am an aspiring AI developer focused on building practical, real-world software with Python, LLMs, automation, and data-driven tools. My work is centered around learning by shipping: small experiments, useful apps, and complete portfolio projects that connect ideas to working systems.

Right now, I am especially interested in:

- AI-powered applications with useful user workflows

- Python automation, Streamlit apps, and backend fundamentals
- Data analytics projects using Python, SQL, Power BI, and Tableau
